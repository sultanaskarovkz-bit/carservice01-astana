# CarService 01 · Astana

Одностраничный сайт СТО **CarService 01** (Астана) в концепции «живой приборной панели / бортового диагностического компьютера». Mobile-first, без внешних зависимостей кроме Google Fonts — весь HTML, CSS и JavaScript в одном файле `index.html`.

## Концепция

Сайт устроен как цифровая панель приборов автомобиля — это напрямую отсылает к их услуге компьютерной диагностики:

- **Загрузка = запуск двигателя:** экран инициализации, стрелка тахометра делает «прогазовку» и уходит на холостые, лампы-предупреждения мигают и гаснут.
- **Услуги = системы авто на диагностике:** каждый блок (двигатель, ходовая, электрика, инжектор, диагностика) при прокрутке «сканируется» и переключается с «неисправность» на «ИСПРАВНО».
- **Живой терминал OBD-II** печатает диагностический протокол приёма.
- **Записи с постов:** лента реальных видео из цеха, оформленная как камеры наблюдения (метка REC, уголки видоискателя). Видео подгружаются лениво и играют только когда попадают в кадр.
- **Рейтинг 4.9** показан как спидометр, статистика набегает как одометр.
- **Нижняя консоль** с кнопками «Позвонить» и «WhatsApp», как центральная консоль в машине.

## Видео

В `video/` лежат 5 роликов, снятых в цеху, отобранных из присланных клиентом записей. Каждый сжат под мобильную загрузку: H.264 (yuv420p), без звука, `+faststart`, суммарно 4.5 МБ вместо исходных 27 МБ. У каждого есть постер `.jpg` — он показывается до загрузки видео.

| Файл | Что на видео | Услуга |
|---|---|---|
| `injector.mp4` | стенд промывки форсунок LAUNCH | инжектор |
| `head.mp4` | снятие головки блока | двигатель |
| `crank.mp4` | разбор двигателя, задиры на коленвале | двигатель |
| `block.mp4` | разбор блока цилиндров | двигатель |
| `engine.mp4` | двигатель в сборе | двигатель |

Пересжать после замены исходника:

```bash
ffmpeg -ss 0 -t 12 -i src.mp4 -vf "scale='min(iw,if(gt(iw,ih),854,480))':-2" \
  -c:v libx264 -crf 30 -preset slow -pix_fmt yuv420p -an -movflags +faststart out.mp4
ffmpeg -ss 0.5 -i out.mp4 -frames:v 1 -q:v 5 out.jpg
```

## Технологии

- Чистый HTML + CSS + JavaScript, один файл, без сборки
- SVG-тахометр и спидометр, анимации на CSS и `requestAnimationFrame`
- IntersectionObserver для анимаций при прокрутке
- Полная поддержка `prefers-reduced-motion`
- Адаптив: mobile-first, ширина контента до 640px

## Форма заявки

Секция «Записаться на диагностику» собирает имя, телефон, авто, услугу и описание проблемы, формирует сообщение и открывает WhatsApp (`wa.me`) с готовым текстом. Бэкенд не нужен — работает на любом статическом хостинге.

## SEO и реклама

- Полный набор мета-тегов, Open Graph/Twitter, микроразметка Schema.org `AutoRepair`, `sitemap.xml`, `robots.txt`, фавиконы.
- Готовые события для аналитики и целей: `cta_diagnostika`, `click_phone`, `click_whatsapp`, `lead_submit`, `section_view` и др.
- Секции размечены якорями (`#uslugi`, `#video`, `#otzyvy`, `#zayavka`, `#kontakty`) для посадочных ссылок в рекламе.
- Подробности настройки — в **`SETUP.md`**.
